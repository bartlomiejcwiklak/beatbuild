is a browser loop-based music making tool, loosely inspired by **incredibox**.


### how to run

you can use the pre-built demo deployment [here](https://bartlomiejcwiklak.github.io/beatbuild/) to skip all the hassle

but, if you'd rather not do that, then:

```bash
npm install && npm run dev
```
will do the trick just as well.


### key features

- **interactive loop engine**: high-performance audio mixing using the web audio api.
- **3d album discovery**: immersive 3d carousel for browsing different musical styles and presets.
- **visual grid control**: intuitive pad-based interface for triggering and mixing loops.
- **keyboard shortcuts**: full keyboard mapping for a professional "beat machine" feel.
- **dynamic animations**: smooth transitions and interactive elements for a premium experience.
- **mobile friendly**: fully responsive design that works across all devices.

### tech stack

- **core:** react 18 + typescript
- **bundler:** vite
- **styling:** vanilla css (custom design system)
- **audio:** web audio api (custom loop engine)
