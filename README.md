is a browser loop-based music making tool, loosely inspired by **incredibox**

### how to run

you can use the pre-built demo deployment [here](https://bartlomiejcwiklak.github.io/beatbuild/) to skip all the hassle

but, if you'd rather not do that, then:

```bash
npm install && npm run dev
```
will do the trick just as well.
